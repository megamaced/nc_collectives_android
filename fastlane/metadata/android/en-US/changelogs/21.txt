F-Droid compatibility: disable the "Dependency metadata" APK signing block that AGP 8.x embeds by default. The block is intended for Play Console reporting; F-Droid's binary scanner flags any non-standard signing block as suspicious, so removing it lets F-Droid ship the upstream signed APK directly (reproducible-build verification, dual-channel interchangeability with GitHub Releases). No user-visible behavioural change.
